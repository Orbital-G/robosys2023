## 必要なソフトウェア
* Python
  * テスト: ver3.7〜3.10

## テスト環境
* Ubuntu-20.04

## plusコマンド
plusを実行後、ターミナルに数字を1行ごと入力してください。
その行に何も入力していない状態でEnterキーを押すとプログラムが終了します。
```
./plus
```
